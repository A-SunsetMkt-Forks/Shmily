# 通过 `root` 将 Android 文件复制到电脑

-   如果手机已 `root` ，通过 `ftp` 复制到电脑

-   没有 `root`

    > 主体思路是通过 `微信/QQ` 等官方数据迁移功能把本机数据迁移到带 `root` 的 `Android` 系统内

    -   借一台有 `root` 的手机
    -   使用 `Android` 模拟器 / 虚拟机 (一般均自带 `root`)
        -   在手机上安装模拟器 / 虚拟机
            -   安卓虚拟机有 vmos 光速虚拟机 等
        -   在电脑上安装模拟器 / 虚拟机
